# Executive Dashboard Mockup

## Purpose

Leadership teams need a simple way to understand the operational health of an event without reviewing multiple spreadsheets or reports.

This dashboard represents the type of high-level information that would support operational decision making.

---

## Example Metrics

| Metric | Value |
|---------|------:|
| Sponsors Confirmed | 42 |
| Attendees Registered | 318 |
| Speakers Confirmed | 27 |
| Sessions Scheduled | 18 |
| Sponsorship Revenue | $186,500 |
| Tasks Complete | 94% |
| Outstanding Issues | 3 |

---

## Executive Questions This Dashboard Answers

- Are registrations on track?
- Are sponsor commitments meeting expectations?
- Are operational tasks falling behind?
- Which areas need leadership attention?
- Is the event ready for launch?
